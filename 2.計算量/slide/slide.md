→ $O$ (ランダウの記号)を用いる

# オーダー記法 (2/2)

- 計算量は基本的にオーダー記法で書く
  1. 一番大きい項のみ残して表記する
    $n! > a^n > n^a > \log n > a$　（$a$ は定数）
  2. 定数倍は無視する

オーダー記法の例）
$$
\begin{align*}
  5n^3+4n^2+100n &\longrightarrow O(n^3)\\
  2^n+n^{100}+10^9n &\longrightarrow O(2^n)\\
\end{align*}
$$

# コードの計算量の調べ方

- $n$ 回のループをする → $O(n)$
- $n$ 回のループの中で $n$ 回のループをする（二重ループ）
  → $O(n^2)$
- [bit全探索](https://drken1215.hatenablog.com/entry/2019/12/14/171657)（$n$ 個の要素について**ある**/**ない**の$2$通りを考える）
  → $O(2^n)$
- $n$ 個の順列を全て調べる → $O(n!)$

```python
n = int(input())

ans = 0
```

```python
print(ans)
```

→ $O(n)$ （$n$ までのループを1回している）

# 公式を使う (2/3)

この公式を使えば、、

$$
\sum_{i=1}^{n} = \frac{1}{2} n(n+1)
$$

```python
# 1~n までの数の和を求める
n = int(input())
ans = n * (n + 1) // 2

print(ans)
```
